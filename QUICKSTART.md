## Troubleshooting

### GUI not opening?

**Check taskbar** - Window might be minimized

**Install tkinter:**
```bash
# Linux:
sudo apt-get install python3-tk

# Mac:
brew install python-tk
```

### Ollama issues?

**Manual install:**
- Windows: https://ollama.com/download/windows
- Mac/Linux: `curl -fsSL https://ollama.com/install.sh | sh`
