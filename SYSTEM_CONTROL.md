## üêõ Troubleshooting

### Application Not Found

**Problem:** "Application 'XYZ' not found"

**Solutions:**
1. Check if app is installed
2. Try full app name: "Google Chrome" instead of "Chrome"
3. Add app path to database manually

### Permission Denied

**Problem:** "Permission denied" errors

**Solutions:**
1. Run JARVIS as administrator (Windows)
2. Use `sudo` for system commands (Mac/Linux)
3. Grant necessary permissions in System Preferences

### Command Not Working

**Problem:** Command not recognized

**Solutions:**
1. Use wake word: "Jarvis, ..."
2. Try alternative phrasing
3. Check supported commands list
4. Use English or Hinglish
